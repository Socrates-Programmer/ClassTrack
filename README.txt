🏫 ClassTrack
ClassTrack es una aplicación diseñada para ayudar a los estudiantes a gestionar su información académica de manera eficiente. 
Permite consultar las asignaturas de cada titulación, acceder a información detallada sobre los profesores y generar informes personalizados.

🚀 Características
✔️ Gestión de asignaturas por titulación
✔️ Consulta de información de profesores
✔️ Generación de informes personalizados
✔️ Interfaz intuitiva y fácil de usar
✔️ Base de datos relacional para mayor eficiencia

🛠️ Tecnologías utilizadas
💻 Lenguaje: C#
🔧 Framework: .NET
🗄️ Base de datos: SQL Lite
🌐 Repositorio: GitHub
